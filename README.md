# Denis Novik Лендинг

Сайт-портфолио **UX/UI дизайнера** — одностраничный сайт с информацией о специалисте, его навыках и работах. Проект выполнен на основе Figma-макета из открытого доступа.

## О проекте

Лендинг представляет UX/UI дизайнера из Минска. Сайт рассказывает о дизайнере, его профессиональных навыках (Photoshop, Illustrator, After Effects, Figma) и демонстрирует портфолио из трёх работ: интернет-магазин, концепт Reebok и лендинг Braun.

## Технологии

`HTML` `CSS` `Flexbox` `Адаптивная вёрстка`

## Что реализовано

- Фиксированная шапка с навигацией и плавным скроллом
- Вступительная секция: имя, специализация, возраст/город и переключатель языка (RU/ENG)
- Блок «О себе» с краткой биографией
- Секция «Навыки» с логотипами программ
- Портфолио: 3 работы с изображениями и подписями-ссылками
- Контактная секция: кнопка отправки сообщения, иконки соцсетей (LinkedIn, Instagram, Behance, Dribble)
- Подключение кастомных шрифтов (Gilroy medium/bold)
